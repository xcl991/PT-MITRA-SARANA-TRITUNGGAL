# NIB PT Mitra Sarana Tritunggal - Website HTML

## Informasi Project

Website berbasis HTML profesional untuk **NIB PT Mitra Sarana Tritunggal**, perusahaan Meta Ads terkemuka di Indonesia.

## File yang Dibuat

### Halaman HTML
- **index.html** - Halaman utama/beranda dengan semua section penting
- **privacy.html** - Halaman Kebijakan Privasi
- **terms.html** - Halaman Syarat & Ketentuan

### Stylesheet
- **styles.css** - CSS lengkap dengan tema Navy Blue profesional

### JavaScript
- **script.js** - Fungsionalitas JavaScript untuk interaktivitas

## Fitur Website

- ✅ Tema warna Navy Blue profesional
- ✅ Tipografi modern dan profesional
- ✅ Background image dari URL yang disediakan

## Cara Mengakses

Website dapat diakses melalui:
- **Homepage**: http://localhost:3000/index.html
- **Privacy**: http://localhost:3000/privacy.html

## Catatan Teknis

- Full HTML tanpa React (sesuai request)
- Semua files berada di folder `public/`
- Menggunakan CSS modern dengan variables
- JavaScript untuk interaktivitas
- Responsive design untuk semua ukuran layar
- Accessible dengan semantic HTML
